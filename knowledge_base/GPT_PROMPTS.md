## üéØ Pro Tips for Using These Prompts

1. **Be Specific**: Add your context to make answers more relevant
   ```
   I'm a PM at a B2B SaaS company with 50 employees. Based on Lenny's Podcast, 
   how should I approach [topic]?
   ```

2. **Request Citations**: Ask for specific episode references
   ```
   [Your question] - Please cite specific episodes and guests.
   ```

3. **Ask for Examples**: Request real-world examples
   ```
   [Your question] - Include specific examples from the transcripts.
   ```

4. **Compare Perspectives**: Get multiple viewpoints
   ```
   [Your question] - Compare how different guests approach this.
   ```

5. **Request Actionability**: Make it practical
   ```
   [Your question] - Provide actionable steps I can take tomorrow.
   ```
